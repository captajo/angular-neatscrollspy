Installation


Install NeatScrollspy from NPM repository using the following in your terminal 


        npm install ngx-neatscrollspy --save


Install in your project


Once installed from npm, you need to add NeatScrollspyModule to your project module, to begin to use it. Import NeatScrollspyModule in your app.module.ts file (or any module, you wish to use it) 

          import { NeatScrollspyModule } from 'ngx-neatscrollspy'; 

          ...... 

          @NgModule({
            declarations: [ ... ],
            imports: [
              ....
              NeatScrollspyModule,
            ]
          })
